# NapCat for Android

将 NapCatQQ 移植到 Android 平台的项目，使用户能够在 Android 设备上运行 NapCat 机器人框架。

## 使用说明

1. 安装 APK 到 Android 设备
2. 启动应用并授予必要权限
3. 点击"Start Service"启动 NapCat 服务
4. 服务将在后台持续运行
